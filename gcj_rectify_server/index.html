🗺️ GCJ Rectify 服务

将 GCJ-02（火星坐标系）地图瓦片实时纠正为 WGS-84 坐标系，
作为 QGIS / Leaflet 等 GIS 客户端的标准 WMTS 数据源使用。

📡 WMTS 端点

WMTS GetCapabilities：
/wmts

瓦片地址格式：
/tiles/{map_id}/{z}/{x}/{y}

在 QGIS/Arcmap 中添加 WMS/WMTS 连接，
填写 URL 为当前服务的 /wmts 地址即可加载。

也可用 Leaflet、OpenLayers 等前端库直接使用。
